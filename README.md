# My_Bash_Scripts 
my personal bash scripts repository.

1. **update_pkgs.sh** : a bash script to update all packages using apt.
2. **process_lpi_objectives.sh** : a script to process a text file to an sqlite3 database.
3. **nbr_connected_users.sh** : number of connected users now.
4. **guess_game.sh** : play a guess game.
5. **Get_http_code.sh** : get the http status of an url.
6. **hoover.sh** : delete file based on an MD5 hash.
